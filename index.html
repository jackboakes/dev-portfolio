Snake

View on GitHub

Snake was built as a personal project to strengthen my understanding of low-level programming and game architecture.

Originally I wrote the game as a console application in C but later added raylib to handle opening a window and graphics. I later refactored the project into modern C++17, applying best practices and using features like std::filesystem and std::string_view.

The game's menu UI is built in immediate mode and keeps track of only one active element. Player input is polled every frame and stored in a circular queue of size two, enabling responsive movements. The high score is saved to a text file using std::fstream, and all sound effects and artwork were created from scratch by me.
